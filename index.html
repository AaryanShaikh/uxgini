My Process

Empathize
I immerse myself in the user’s world through interviews, surveys, focus groups, and competitor analysis to understand their needs and pain points, building empathy as the foundation of user-centered design.
Define
Using insights from the Empathize phase, I define the problem and project goals, creating user personas, empathy maps, and journey maps to turn data into clear, actionable insights that guide the design process.
Ideate
In the Ideate phase, I foster creativity and collaboration through brainstorming, mind mapping, card sorting, user flows, information architecture, and sketching to explore diverse design possibilities.
Design
In the Design phase, I turn selected ideas into paper wireframes and high-fidelity prototypes using tools like Figma. I focus on details like typography, color schemes, and visual hierarchy to create polished, effective designs.
Test
Testing is a continuous part of the design process. I use methods like A/B testing, usability surveys, and eye-tracking to gather user feedback and data, ensuring the design meets user needs and expectations.
